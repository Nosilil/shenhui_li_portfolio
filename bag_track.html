BagTrack

An tracking system for your daily items

Skills used

Analogue electronics, digital signal processing, C/C++.

Concept

This project aims to solve the problem of people accidentally leaving things behind.

The device consists of 2 parts, the backpack and the tags. The sticker-like tag can be attached to the item the user wants to track. Every time the item is taken out, or put in, the microcontroller records the timestamp and the current location. It'd alarm the user when the item is left behind. User can also check what's in the bag via the app.

Shenhui Li 2020
